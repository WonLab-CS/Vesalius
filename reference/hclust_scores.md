# Hiearchal clustering of mapping scores for co mapping events

Hiearchal clustering of mapping scores for co mapping events

## Usage

``` r
hclust_scores(score, h, k, verbose)
```

## Arguments

- score:

  matrix containing mapping scores

- h:

  numeric representing the normalized height that will be used for
  clustering (normalized to ensure that it will be between 0 and 1)

- k:

  int - number of clusters

- verbose:

  logical - print progress messages
