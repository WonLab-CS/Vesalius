# check cost validity

check cost validity

## Usage

``` r
check_cost_validity(
  cost,
  seed_assay,
  seed_signal,
  query_assay,
  query_signal,
  use_cost,
  verbose = TRUE
)
```

## Arguments

- cost:

  cost matrix

- seed_assay:

  seed assay

- seed_signal:

  seed signal

- query_assay:

  query assay

- query_signal:

  query signal

- use_cost:

  cost types to use

- verbose:

  logical - progress messages

## Value

validated cost information
