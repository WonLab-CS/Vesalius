# concat cost - pairwise sum of score complement

concat cost - pairwise sum of score complement

## Usage

``` r
concat_cost(cost, use_cost, complement = TRUE)
```

## Arguments

- cost:

  list - named list contained score matrices

- use_cost:

  character - which cost matrices to use

- complement:

  logical - whether to use complement of scores

## Value

list with cost matrix
