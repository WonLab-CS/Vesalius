# check if image is present

check if image is present

## Usage

``` r
check_image(vesalius_assay, image_name = NULL, as_is = FALSE)
```

## Arguments

- vesalius_assay:

  vesalius assay object

- image_name:

  character - name of the image to retrieve

- as_is:

  logical - return image as is

## Value

microscopy image
