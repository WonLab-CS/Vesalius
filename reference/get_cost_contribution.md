# Compute mapping score contribution to mapping

Compute mapping score contribution to mapping

## Usage

``` r
get_cost_contribution(vesalius_assay, method = "dispersion", verbose = TRUE)
```

## Arguments

- vesalius_assay:

  vesalius_assay object after mapping a query onto a reference assay.

- method:

  character - which method to use to compute contribution currently only
  dispersion availble

- verbose:

  logical - print progress messages
