# dispatch territories labels territories according to which group they belong to.

dispatch territories labels territories according to which group they
belong to.

## Usage

``` r
dispatch_territory(territories, ter_1, ter_2, cells)
```

## Arguments

- territories:

  territories data frame from territoires slot in vesalius_assay object

- ter_1:

  integer vector containing territories in group 1

- ter_2:

  integer vector containing territories in group 2

- cells:

  cell barcodes
