# dispatch barcodes associted with a specific territory in a specific samples

dispatch barcodes associted with a specific territory in a specific
samples

## Usage

``` r
dispatch_sample(territory_barcodes, ter, sample)
```

## Arguments

- territory_barcodes:

  character vector barcodes associated with territory

- ter:

  chracter territory id

- sample:

  character vector associeted with sample

## Value

character vector of barcodes at the intersection of territory and sample
