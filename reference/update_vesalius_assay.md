# update vesalius assay object

update vesalius assay object

## Usage

``` r
update_vesalius_assay(vesalius_assay, data, slot, append = TRUE)
```

## Arguments

- vesalius_assay:

  a vesalius_assay object

- data:

  data that will be inserted into the vesalius_assay

- slot:

  name of the slot that will be updated

- append:

  logical if the dats hould be appended to already existing data

## Value

a vesalius_assay
