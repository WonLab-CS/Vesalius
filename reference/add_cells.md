# add cells to vesalius assay

add cells to vesalius assay

## Usage

``` r
add_cells(vesalius_assay, cells, add_name = NULL, verbose = TRUE)
```

## Arguments

- vesalius_assay:

  vesalius_assay object

- cells:

  data.frame containing cell coordinates and metadata

- add_name:

  character string defining trial name for added cells

- verbose:

  logical - progress message output

## Value

vesalius_assay object with added cells
