# commits function call to log slot

commits function call to log slot

## Usage

``` r
commit_log(vesalius_assay, commit, assay)
```

## Arguments

- vesalius_assay:

  a vesalius_assay object

- commit:

  list containing arguments used in latest function call

- assay:

  assay name used in latest function call

## Value

vesalius_assay with updated log slot
