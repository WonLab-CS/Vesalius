# convert cimg list to vesalius object embedding

convert cimg list to vesalius object embedding

## Usage

``` r
format_c_to_ves(cimg, vesalius_assay, dims, embed = "last", verbose = TRUE)
```

## Arguments

- cimg:

  cimg image list

- vesalius_assay:

  a vesalius_assay object

- dims:

  integer vector of embedding that need to be updated

- embed:

  character string which embedding should be updated

- verbose:

  logical should progress message be outputed

## Value

a vesalius_Assay obejct
