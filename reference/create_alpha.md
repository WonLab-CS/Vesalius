# create alpha value if territories need to be highlighted

create alpha value if territories need to be highlighted

## Usage

``` r
create_alpha(territories, highlight, alpha)
```

## Arguments

- territories:

  vesalius territories taken from a vesalius_assay

- highlight:

  numeric vector describing which territories should be highlighted

- alpha:

  tranaparent factor

## Value

vector of alpha values

## Details

If highlight is null, will return the same alpha values for all
territories
